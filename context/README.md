# context

the original scripts and documentation from before the restructure. kept here as reference — these are what everything in `scripts/` was ported from.

not meant to be run directly. read them to understand the lineage of the current code, or to port remaining features that haven't been extracted yet.

## what's here

- **scripts/blend/** — the original blend scripts (blend-video.py, blend-video-alt.py, etc.)
- **scripts/ai/** — stable diffusion and ML-based video experiments
- **scripts/post/** — analog processing, stabilization, vertical conversion
- **scripts/upload/** — youtube upload scripts (simple_youtube_api based)
- **scripts/utils/** — audio extraction, loop extension, rembg tests
- **scripts/text/** — project name generator
- **COLOR-KEYING-APPROACHES.md** — notes on different keying strategies
- **HOW-IT-WORKS.md** — original workflow documentation
- **REFACTOR-PLAN.md** — early planning notes for the restructure
